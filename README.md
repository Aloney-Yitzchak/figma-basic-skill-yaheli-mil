# משימה 3: עיצוב אפליקציה ב-Figma 🎨

## מטרת המשימה
ללמוד לעצב ממשק משתמש (UI/UX) באמצעות Figma לפני כתיבת קוד. תעצבו אפליקציית "רשימת קניות" עם 3 מסכים ותיצרו Prototype אינטראקטיבי.

---

## בדיקות אוטומטיות 🤖

המשימה כוללת בדיקות אוטומטיות שרצות בכל פעם שאתם עושים push.

| בדיקה | ניקוד | מה נבדק |
|-------|-------|---------|
| קבצים נדרשים | 50 | DESIGN.md קיים + לפחות 2 תמונות בתיקיית screenshots |
| שלמות התוכן | 50 | DESIGN.md מכיל לפחות 20 שורות |
| **סה"כ** | **100** | |

### איך לבדוק את התוצאות:
1. עשו Push לקוד שלכם
2. לכו ללשונית **Actions** בריפוזיטורי
3. לחצו על ה-workflow האחרון
4. ✅ = הבדיקה עברה, ❌ = צריך לתקן

**טיפ:** אפשר לעשות push כמה פעמים שרוצים! הבדיקות ירוצו שוב בכל פעם.

---

## הוראות

### חלק א' - יצירת חשבון Figma (0 נקודות - הכנה)

1. היכנסו ל-[figma.com](https://www.figma.com)
2. הירשמו עם חשבון Google של בית הספר
3. בחרו בתוכנית החינמית (Starter)
4. צרו קובץ עיצוב חדש (New Design File)

---

### חלק ב' - עיצוב 3 מסכים (60 נקודות)

עצבו אפליקציית **"רשימת קניות"** עם 3 מסכים:

#### מסך 1: התחברות (Login) - 20 נקודות
- [ ] לוגו או שם האפליקציה
- [ ] שדה שם משתמש (EditText)
- [ ] שדה סיסמה (EditText)
- [ ] כפתור "התחבר" (Button)
- [ ] מרווחים אחידים בין האלמנטים

#### מסך 2: רשימת קניות (Main) - 20 נקודות
- [ ] כותרת "רשימת הקניות שלי"
- [ ] רשימת פריטים (לפחות 3 פריטים לדוגמה)
- [ ] כפתור "הוסף פריט" (בתחתית או כ-FAB)
- [ ] כל פריט ברשימה מציג: שם + כמות + checkbox

#### מסך 3: הוספת פריט (Add Item) - 20 נקודות
- [ ] כותרת "הוסף פריט חדש"
- [ ] שדה שם פריט
- [ ] בחירת כמות (מספר או +/-)
- [ ] כפתור "שמור"
- [ ] כפתור "ביטול" או חץ חזרה

**דרישות עיצוב לכל המסכים:**
- גודל Frame: Android Large (360x800) או דומה
- פלטת צבעים אחידה (עד 3 צבעים עיקריים)
- גופן אחיד (מומלץ: Roboto)
- מרווחים בכפולות של 8px

---

### חלק ג' - Prototype (20 נקודות)

הוסיפו מעברים אינטראקטיביים:

1. [ ] כפתור "התחבר" → מעבר למסך רשימת הקניות
2. [ ] כפתור "הוסף פריט" → מעבר למסך הוספת פריט
3. [ ] כפתור "שמור" / "ביטול" → חזרה למסך רשימת הקניות

**טיפ:** עברו ללשונית Prototype, בחרו כפתור, וגררו את העיגול הכחול למסך היעד.

---

### חלק ד' - תיעוד (20 נקודות)

1. **צלמו screenshots** של כל 3 המסכים ושמרו בתיקיית `screenshots/`
   - שם קובץ מומלץ: `01-login.png`, `02-main.png`, `03-add-item.png`

2. **מלאו את `DESIGN.md`** - הסבירו את בחירות העיצוב שלכם

3. **שתפו קישור ל-Figma** - בתוך DESIGN.md, הדביקו את קישור השיתוף של הפרויקט

## בעיות נפוצות

**בעיה:** לא יודע איך לצלם screenshot ב-Figma
**פתרון:** בחרו את ה-Frame ← לחצו על שלוש הנקודות למעלה ← Export ← PNG ← Export

**בעיה:** Prototype לא עובד
**פתרון:** ודאו שאתם בלשונית Prototype (לא Design), ושהחיצים מחוברים נכון

**בעיה:** הצבעים לא אחידים
**פתרון:** הגדירו Color Styles ב-Figma והשתמשו בהם בכל המסכים
